# 🥕 GitRabbit Frontend

The user-facing dashboard and web application interface for GitRabbit, built with Next.js, React, Tailwind CSS, and Framer Motion.

For a full overview of the project, architecture, features, and engine capabilities, please refer to the main **[Root README.md](../README.md)**.

## 🚀 Dev Commands

From inside this directory (`/frontend`):

| Command | Description |
|:---|:---|
| `npm run dev` | Runs the Next.js development server with Turbopack support |
| `npm run build` | Compiles the production application bundle with TypeScript safety check |
| `npm run start` | Boots the compiled production server |
| `npm run lint` | Runs ESLint utility checks on the codebase |

## 🛠️ Tech Stack & Key Libraries

- **Framework:** Next.js (using modern App Router)
- **State/Theme:** Context-based system (`ThemeContext`)
- **Animation:** Framer Motion (pixel-art micro-animations and physics)
- **Styling:** Vanilla CSS variables + Tailwind Utility classes
- **Icons:** Lucide React icons
